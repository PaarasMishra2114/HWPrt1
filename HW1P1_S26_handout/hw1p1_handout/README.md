# HW1P1 Handout

This folder contains the handout code and tests for HW1P1.

## Contents
- `mytorch/`: Minimal neural network library implementation.
- `models/`: Model definitions.
- `autograder/`: Autograder scripts and flags.
- `test_*.py`: Unit tests.
- `requirements.txt`: Python dependencies.
- `create_tarball.sh`: Helper script to package your submission.

## Quick start
1. Create and activate a Python environment.
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Run tests:
   - `python -m pytest`

## Notes
- Use `show_results.py` to view autograder output if provided.
- Do not modify autograder files unless instructed.
